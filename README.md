Masker

Программа на Go, которая читает текстовый файл, заменяет все цифры в строках на символ * и сохраняет результат в другой файл.

Установка

1. Убедитесь, что установлен Go версии 1.21 или выше.
2. Склонируйте репозиторий и перейдите в папку:

git clone https://github.com/qwaseri832/masker.git
cd masker

3. Скачайте зависимости:

go mod tidy

Использование

Запуск программы:

go run main.go input.txt output.txt

где:
- input.txt – путь к исходному файлу (обязательный аргумент)
- output.txt – путь к результату (необязательный, по умолчанию output.txt)

Пример

input.txt:

Привет 123
Мой телефон 89123456789
Год 2025

Запуск:

go run main.go input.txt result.txt

result.txt:

Привет ***
Мой телефон *************
Год ****

Если файл уже существует

Программа перезапишет выходной файл.

Тестирование

Запуск тестов с покрытием:

go test -cover ./masker/

Ожидаемый результат: покрытие > 85%

Структура проекта

masker/
├── go.mod
├── go.sum
├── main.go
├── masker/
│   ├── producer.go       // читает файл

│   ├── service.go        // бизнес-логика (маскирование)
│   └── *_test.go         // тесты
└── README.md

- Go 1.21+
